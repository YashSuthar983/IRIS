# IRis Website

This directory contains the source code for the IRis website, split into frontend and backend.

## Structure

- **frontend/**: A Next.js application for the user interface.
- **backend/**: A generic backend (likely Python/Flash based on file exploration) to serve the API.

## Getting Started

### Frontend
Navigate to the `frontend` directory:
```bash
cd frontend
npm install
npm run dev
```

### Backend
Navigate to the `backend` directory:
```bash
cd backend
# Setup your python environment
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
# Run the application
python app.py
```
